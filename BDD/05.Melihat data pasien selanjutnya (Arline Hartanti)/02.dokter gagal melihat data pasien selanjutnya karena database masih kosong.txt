Fitur	        Melihat data pasien selanjutnya
Skenario	Dokter gagal melihat data pasien selanjutnya karena database antrian 
                untuk dokter belum terisi (masih kosong)
 		

Given 		Dokter masuk ke halaman khusus dokter
Then		Sistem mengirimkan data pasien ke dokter
And 		Sistem menampilkan field data pasien selanjutnya masih belum terisi (kosong) 
And		Sistem menampilkan pesan dialog box bahwa antrian selanjutnya masih kosong ,silahkan tunggu ... 
When		Petugas meng-klik tombol "OK"
Then		Sistem mengulang proses menampilkan pasien selanjutnya
